  • file
  • isTrack
  • autoPlay
  • callback
)

Defined in gkSound.js:239

Load sound. Simple away to add sound in your page. Supports MP3 and OGG files.

Parameters:

  • id String

    Id to new Audio element.

  • file String

    File name can be with or without extension name.

  • isTrack Boolean

    Boolean default false. Set to true to play sound with loop.

  • autoPlay Boolean

    Boolean default false. Set to true to play sound when is loaded.

  • callback Function

    Callbacks return an Object like { message:"Sound loaded.", code:0 }. These codes are according HTML5 specifications.

)

Defined in gkSound.js:365

Load sound. Simple away to add sound in your page. Supports MP3 and OGG files.

Parameters:

  • id String

    Id of Audio element.

  • override Boolean

    Boolean default false. Set to true to play sound from start.

  • callback Function

    Callbacks return an Object like { message:"Sound played.", code:0 }. Code will be -1 if sound not played.

  • ignoreMute Boolean

    Boolean default false. Set to true to play sound ignoring mute settings.
